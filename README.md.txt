#🗑 SMART TRASH CAN TC1 
This is a the source code 📄 to deploy on the TC1 with the following features:
#🕹 Features 
* Auto open the lid when a hand hover on top (from 10 to 15CM 📏)
* Auto close or remain open the lid when the user is still infront of it or not (⚠ Error, currently the door is not perpendicular with the ground so the inner Ultra Sonic sensor cannot detect the front user)
* Calulate ⏱ the amount of remaining trash inside the can
#🗃 Libraries
* Arduino Servo - [](https://github.com/arduino-libraries/Servo)
#⚙ Components
* Arduino ESP32
* Servo MG996R
* Ultrasonic HC-SR04
